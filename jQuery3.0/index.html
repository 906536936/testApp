jQuery3.0部分变化

变化

  1. unwrap方法
  2. data() 将所有的属性key变成驼峰形式,在 jQuery 3 中，属性名已经变成了驼峰形式，横杠已经被去除了；而在以前的版本中，属性名会保持全小写，并原样保留横杠
  3. bind()、delegate()、unbind() 和 undelegate() 已经不再推荐使用
  4. 移除 load()、unload() 和 error() 方法
  5. jQuery 对象移除context、support 和 selector 属性
  6. jQuery 对象支持for of语法遍历，如 var ss = $("body") for(var v of ss){console.log(v)}，以前版本不支持
  7. 彻底放弃对 IE8 的支持

fixed bugs

  1. width(),height() 不再舍弃小数，而是返回浮点数值
1.unwrap：请查看dom结构
2.data:请查看key值
